# 1.条件判断中的x

​	在shell脚本中，经常看到`if [ "x${var}" == "x" ]`这种判断条件。其作用是防止出现语法错误。如果不写x，只用`if [ "${var}" == "0" ]`来判断${var}的值，当${var}为空或未设置时，语句被解释为`if [ == "0" ]`，出现语法错误。加上x后，当${var}为空或未设置时，解释为`if [ "x" == "x" ]`，语法上依然正确。

​	`if [ "x${var}"=="x" ]`整句的意思是判断${var}是否为空。
